# Zoom-Out — 模块全景地图

在不熟悉的代码区域提升一个抽象层级。给出相关模块和调用者的全景地图，使用项目用语。

## 输出格式

```markdown
## {模块名} 全景
**职责**：{一句话}
**依赖关系**：
- 调用方：{谁调用了这个模块}
- 被调用方：{这个模块调用了谁}
**关键入口**：{对外暴露的接口/函数}
**相关用语**：{涉及的项目用语}
```

## 停止条件

- 全景地图输出完毕 → 停止
