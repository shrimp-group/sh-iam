# Handoff — 会话交接

将当前对话压缩为一份交接文档，让新的 agent 会话能无缝继续工作。

## 输出位置

保存到系统临时目录，不保存到工作区。
文件名格式：`handoff-YYYYMMDD-HHmm.md`

## 文档结构

```markdown
# 交接文档
**生成时间**：YYYY-MM-DD HH:mm
**下次会话重点**：[用户传入的参数，或自动生成]

## 背景
[1-2 段话，描述当前项目状态和正在做的事]

## 已完成
- [列出本次会话完成的关键动作]

## 未完成
- [列出待办事项，标注优先级]

## 关键决策
| # | 决策 | 理由 |
|---|------|------|
| 1 | ... | ... |

## 建议技能
[根据上下文动态生成]

## 参考资料
- [链接到相关文件、commit、issue]
```

## 规则

1. **引用而非重复** — 已有产物只引用路径/URL，不复制内容
2. **脱敏** — 移除 API key、密码、token、个人信息
3. **聚焦** — 围绕用户传入的重点组织文档
4. **可执行** — 下一个 agent 读完文档后能直接继续工作

## 停止条件

- 交接文档生成完毕 → 停止
