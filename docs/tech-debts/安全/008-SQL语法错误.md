# TD-008: IamApiMapper SQL 语法错误

## 元信息

| 字段   | 值                  |
|------|--------------------|
| ID   | TD-008             |
| 分类   | SEC / code-quality |
| 严重程度 | low                |
| 状态   | open               |
| 发现日期 | 2026-06-19         |

## 描述

IamApiMapper.xml 中 getApis4Paste 查询 SQL 语法错误：deleted = 0 和 api_code IN 之间缺少 AND 关键字，运行时将报 SQL
语法错误。该功能（API 粘贴）无法正常运行。

## 影响范围

- iam-admin (IamApiMapper.xml)
- API 粘贴功能

## 复现条件

1. 调用 API 粘贴接口
2. 执行 getApis4Paste SQL
3. SQL 语法错误导致查询失败

## 当前解决方案

无。功能完全不可用。

## 建议解决方案

在 deleted = 0 后添加 AND 关键字：

```sql
AND deleted = 0
AND api_code IN
```

## 关联模块

iam-admin (IamApiMapper.xml)
