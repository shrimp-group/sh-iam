# TD-015: LoggingFilter.isLog() 使用 synchronized

## 元信息

| 字段   | 值                 |
|------|-------------------|
| ID   | TD-015            |
| 分类   | PERF / bottleneck |
| 严重程度 | low               |
| 状态   | open              |
| 发现日期 | 2026-06-19        |

## 描述

LoggingFilter.isLog() 方法使用 synchronized 关键字，高并发下成为瓶颈。每次请求都需要获取锁来判断是否记录日志。

## 影响范围

- iam-sdk (LoggingFilter)
- 所有经过 LoggingFilter 的请求

## 复现条件

1. 高并发请求场景
2. 所有请求线程争抢 synchronized 锁
3. 吞吐量下降

## 当前解决方案

使用 synchronized 保证线程安全。

## 建议解决方案

改用 ConcurrentHashMap 的 computeIfAbsent 原子操作，避免全局锁。

## 关联模块

iam-sdk (LoggingFilter)
