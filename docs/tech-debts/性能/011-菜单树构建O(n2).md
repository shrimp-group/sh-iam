# TD-011: 菜单树构建使用 O(n^2) 算法

| ID   | TD-011            |

| 严重程度 | medium            |

## 描述

SsoResourceService.makeTree() 使用 O(n^2) 双层循环构建菜单树，菜单数量大时性能差。而 IamMenuService.buildMenuTree() 使用
HashMap 一次遍历构建，效率更高。

## 影响范围

- iam-sso (SsoResourceService)
- 用户菜单查询接口

## 复现条件

1. 菜单数量达到数百或数千
2. 每次菜单查询都需要 O(n^2) 遍历
3. 响应时间随菜单数量平方级增长

## 当前解决方案

功能正确但性能差。

## 建议解决方案

改为 HashMap 一次遍历构建，与 IamMenuService.buildMenuTree() 实现一致。

iam-sso (SsoResourceService)
