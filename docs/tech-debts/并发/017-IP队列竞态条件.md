# TD-017: IP 队列 contains + offer 非原子操作

## 元信息

| 字段   | 值                     |
|------|-----------------------|
| ID   | TD-017                |
| 分类   | CONC / race-condition |
| 严重程度 | medium                |
| 状态   | open                  |
| 发现日期 | 2026-06-19            |

## 描述

IpLocalCacheHelper.offerQueue() 中 IP_QUEUE.contains(remoteAddr) 是 O(n) 操作，且与后续 offer 非原子，高并发下可能重复入队。

## 影响范围

- iam-common (IpLocalCacheHelper)

## 复现条件

1. 高并发下多个线程同时检查同一 IP
2. contains 返回 false 后，多个线程同时 offer
3. 同一 IP 被多次入队

## 当前解决方案

使用 ConcurrentLinkedQueue + contains 检查。

## 建议解决方案

使用 ConcurrentHashSet 替代 contains 检查，或使用 putIfAbsent 语义。将 Set 和 Queue 合并为一个原子操作。

iam-common (IpLocalCacheHelper)
