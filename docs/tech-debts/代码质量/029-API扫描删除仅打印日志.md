# TD-029: RestfulScan 删除 API 仅打印日志

## 描述

RestfulScan 中删除 API 仅打印日志不执行删除，且日志中包含 SQL 拼接（有注入风险提示）。API 自动扫描发现已删除的路由时，仅打印日志但不清理数据库。

## 影响范围

- iam-admin (RestfulScan)
- API 自动扫描功能

## 复现条件

1. 删除代码中的 @Router 注解路由
2. 重启应用
3. RestfulScan 发现差异但仅打印日志
4. 数据库中残留无效 API 记录

## 当前解决方案

仅打印日志。

## 建议解决方案

1. 实现安全的删除逻辑（使用参数化查询）
2. 或明确标注为手动操作，提供管理界面清理

## 关联模块

iam-admin (RestfulScan)
