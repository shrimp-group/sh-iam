# TD-030: AkSignHelper 注释代码不可编译

## 元信息

| 字段   | 值                    |

| ID   | TD-030               |
| 分类   | OTHER / code-quality |
| 严重程度 | low                  |

## 描述

AkSignHelper 中 RedisLock 依赖被注释，但 deSign() 方法中仍引用 redisLock.lock()，代码处于不可编译状态。如果取消注释将无法编译。

## 影响范围

- iam-sdk (AkSignHelper)

## 复现条件

1. 取消 deSign() 方法的注释
2. 编译失败，redisLock 未注入

## 当前解决方案

deSign() 方法被完整注释。

## 建议解决方案

要么完整实现 deSign()（包括 RedisLock 注入），要么清理注释代码。

iam-sdk (AkSignHelper)
