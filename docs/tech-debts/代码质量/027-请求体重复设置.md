# TD-027: IamRequestService 中 requestBody 重复设置

## 元信息

| 字段   | 值                    |
|------|----------------------|
| ID   | TD-027               |
| 分类   | OTHER / code-quality |
| 严重程度 | medium               |
| 状态   | open                 |
| 发现日期 | 2026-06-19           |

## 描述

IamRequestService.insertLog() 中 requestBody 被设置了两次，第二次 setRequestBody 会覆盖第一次的脱敏结果，导致脱敏处理失效。

## 影响范围

- iam-sso (IamRequestService)
- 请求日志中的敏感信息

## 复现条件

1. 请求日志中包含密码等敏感信息
2. 第一次 setRequestBody 进行了脱敏
3. 第二次 setRequestBody 覆盖了脱敏结果
4. 敏感信息被明文存储

## 当前解决方案

无。

## 建议解决方案

删除重复的 setRequestBody 调用，保留脱敏后的设置。

## 关联模块

iam-sso (IamRequestService)
