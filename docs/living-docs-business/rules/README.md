# 业务规则索引

最后更新：2026-06-19

## 规则列表

| 规则名      | 规则描述                           | 适用范围      |
|----------|--------------------------------|-----------|
| 密码加密规则   | 密码使用 MD5(password + salt) 加密存储 | 用户创建/密码修改 |
| 密码过期规则   | 密码默认 180 天过期                   | 登录时检查     |
| 验证码触发规则  | 1 小时内登录失败次数达到阈值时需要验证码          | 登录        |
| JWT 过期规则 | JWT 默认 24 小时过期，支持时钟偏差容忍        | 鉴权        |
| 逻辑删除规则   | 所有表使用 deleted 字段逻辑删除           | 全局        |
| 乐观锁规则    | 所有表使用 version 字段乐观锁            | 全局        |
| 用户名唯一规则  | 用户名全局唯一                        | 用户创建/修改   |
| 角色树校验规则  | 删除角色时校验是否有子角色                  | 角色删除      |
| 菜单树校验规则  | 删除菜单时校验是否有子菜单                  | 菜单删除      |
| 角色有效期规则  | 用户-角色关联含有效期，过期自动失效             | 用户角色绑定    |
