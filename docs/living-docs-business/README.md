# sh-iam 业务活文档

最后更新：2026-06-19

## 业务领域概述

sh-iam 覆盖身份认证与访问管理 (IAM) 领域，核心职责包括：用户身份认证、访问授权、菜单资源管理、应用隔离管理、密钥管理和审计日志。
