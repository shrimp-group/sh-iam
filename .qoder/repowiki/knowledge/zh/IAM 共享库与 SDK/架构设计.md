- **iam-common**：作为基础数据层，定义了 IAM 核心实体（如 User, Role, Tenant）及其 DTO，并提供 `PasswordHelper` 等静态工具类处理密码哈希与历史校验。
- **iam-sdk**：作为客户端集成层，通过 Spring Boot AutoConfiguration (`IamSdkAutoConfig`) 实现自动装配。核心包含 `IamAuthFilter` 拦截请求进行 JWT 校验与 Session 同步，`SsoFacade` 提供日志记录等扩展接口，以及 `JwtUtil`、`SessionHelper` 等安全组件。
- **依赖方向**：SDK 依赖 common 模块获取基础模型，应用系统通过引入 SDK 即可接入统一的 IAM 认证体系。
