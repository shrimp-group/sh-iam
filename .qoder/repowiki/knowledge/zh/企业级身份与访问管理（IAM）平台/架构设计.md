- **分层模块化架构**：采用 Maven 多模块结构，`iam-common` 提供跨模块共享的实体与 DTO，`iam-sdk` 封装 JWT 鉴权过滤器与 SSO 门面模式，供 `iam-sso` 和 `iam-admin` 依赖。
- **前后端分离部署**：包含两套独立的前端应用（`iam-admin-ui` 管理端、`iam-sso-ui` 用户门户）与对应的后端 Starter 启动器，通过 Nginx 或网关进行路由分发。
- **统一安全契约**：所有后端模块遵循统一的 RESTful 路由规范（`/iam-sso/**` 与 `/iam-admin/**`），并通过 `iam-sdk` 中的 `IamAuthFilter` 实现基于 JWT 和 Redis Session 的分布式会话校验。
- **自动化配置集成**：利用 Spring Boot AutoConfiguration 机制，各功能模块（如 Admin、SSO、SDK）通过 `META-INF/spring/org.springframework.boot.autoconfigure.AutoConfiguration.imports` 实现即插即用。
