- API 模块按业务域（system, user, log）划分目录，每个文件导出对应的请求方法，统一使用 `@/utils/request` 实例。
- 业务页面采用“列表页 + 编辑组件”的模式，列表页位于 `views/[module]/index.vue`，弹窗编辑逻辑封装在同级 `components/edit.vue` 中。
- 全局工具函数（如时间格式化、树形数据转换）挂载在 `app.config.globalProperties` 上，以便在模板中直接调用。
- Element Plus 组件在全局层面进行了默认属性配置，如表格默认开启边框和溢出提示，弹窗默认追加至 body 且支持拖拽。
