- **技术栈**：采用 Vue 3 (Composition API) + Vite 6 + Pinia + Vue Router 4 + Element Plus。
- **分层架构**：遵循经典的前端分层模式，`src/api` 负责后端接口封装，`src/store` (Pinia) 管理全局状态（用户、权限、字典），`src/views` 承载业务页面，`src/components` 提供通用 UI 组件。
- **权限控制**：实现基于 RBAC 的动态路由机制。通过 `src/permission.js` 全局守卫拦截请求，登录后调用 `userMenuTree` 接口获取菜单树，动态注册路由并渲染侧边栏。
- **交互增强**：在 `src/main.js` 中批量注册全局组件（如分页、文件上传）和工具方法；通过自定义指令 `v-hasPermi` 和插件 `auth.js` 实现细粒度的按钮级权限控制。
- **网络层**：基于 Axios 封装 `request.js`，统一处理 Token 注入、重复提交校验、错误码映射及 Blob 下载逻辑。
