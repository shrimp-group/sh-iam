- **技术栈**: 采用 Vue 3 (Composition API) + Vite 构建，UI 框架为 Element Plus，状态管理使用 Pinia。
- **分层结构**: `src/api` 封装 SSO 认证与用户信息接口；`src/store` (Pinia) 管理用户会话与权限状态；`src/views` 包含登录、注册、个人中心及门户仪表盘页面；`src/components` 提供大量通用业务组件（如文件上传、富文本、字典标签等）。
- **安全与路由**: `src/permission.js` 实现全局路由守卫，结合 JWT Token 进行身份校验与动态路由加载；`src/utils/request.js` 配置 Axios 拦截器，处理 Token 注入、重复提交防护及统一错误响应。
- **样式与资源**: 使用 SCSS 进行样式管理，集成 SVG 图标系统，支持深色模式切换。
