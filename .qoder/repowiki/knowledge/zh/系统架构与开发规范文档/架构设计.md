- **分层架构**：采用表现层（Vue3）、API网关层（Spring Boot Rest）、业务逻辑层（Admin/SSO/SDK）和数据访问层（MyBatis + MySQL/Redis）的经典四层设计。
- **模块职责**：`iam-common` 负责基础实体与工具，`iam-sdk` 处理 JWT 鉴权与 AK 签名，`iam-sso` 承载登录注册等公开业务，`iam-admin` 管理用户、角色、菜单等核心资源。
- **数据持久化**：基于 MySQL 8.0+ 存储业务数据，采用逻辑删除与乐观锁机制；使用 Redis 7.0+ 管理 Session、验证码及高频缓存。
- **安全体系**：实施 JWT + Redis 双重 Token 验证，密码采用 RSA 传输加密与 MD5+salt 存储，并通过 `IamAuthFilter` 实现统一的请求鉴权与日志记录。
