- 采用 Spring Boot Starter 架构模式，`iam-admin` 为自动配置库，通过 `IamAdminAutoConfig` 和 `META-INF/spring.factories` 实现组件扫描与 Mapper 注册；`iam-admin-starter` 为独立可执行入口。
- 遵循经典三层架构：`rest` 层处理 HTTP 请求并定义路由（`Route.java`），`service` 层封装业务逻辑（如用户创建时的多表事务处理），`mapper` 层配合 XML 实现数据持久化。
- 具备启动时 API 自动同步机制（`RestfulScan`），通过对比运行时 Rest 映射与数据库记录，自动维护 API 权限元数据。
- 依赖 `iam-sso` 模块实现单点登录集成，使用 MyBatis 进行 ORM 映射，Redis 用于 ID 生成。
